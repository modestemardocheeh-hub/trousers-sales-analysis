# Trousers Sales Analysis

## Description
Complete analysis of a clothing sales dataset using Python, Pandas and Matplotlib.
This project covers data cleaning, descriptive statistics, product analysis,
payment analysis, time series analysis and data visualization.

## Dataset
- 116 rows of sales transactions
- Products : 13 different trouser models
- Period : August to October 2025
- Payment methods : Cash, Credit Card, Debit Card, eWallet

## Features
- Data cleaning and preparation
- Descriptive statistics (mean, median, std, min, max)
- Product performance analysis (top 5 best sellers, top 5 most profitable)
- Payment method analysis with percentages
- Time series analysis (by day, week, month)
- Order analysis (multi-item orders, average order value)
- Pivot tables (products x payment methods)
- 8 visualizations (bar charts, pie chart, line charts, histograms)
- Excel export with 5 sheets

## Technologies
- Python 3
- Pandas
- Matplotlib
- XlsxWriter

## Key Findings
- Total revenue : calculated from 116 transactions
- Most popular payment method : identified via value_counts()
- Best selling product : identified via groupby analysis
- Peak sales month : identified via temporal analysis
